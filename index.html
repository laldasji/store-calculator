The following calculator evaluates expressions in the form of a stack. That is, expressions are evaluated as they are entered, and not using the rule BODMAS, or PEMDAS.
For example, the expression:

10 + 45 * 2


would NOT be evaluated as

⇒ 10 + 45 * 2
= 10 + (45 * 2)
= 10 + 90
= 100


Instead, it will be evaluated as follows:

⇒ 10 + 45 * 2
= 55 * 2
= 110


Expressions will be evaluated left to right, as they are input into the calculator

A Brief Description of unique buttons:

  • AC: clears all previous calculations
  • C: works as a "backspace", clears only the last character.
  • EXP: exponent, as:

    result operand

  • %: modulus operator, returns remainder
  • "=": Returns the final answer of the expression. Only press when further operations are NOT required.
Keyboard Controls
  • Regular Controls for all numbers, be it via the numpad or main numeric keys
  • All operations done via the corresponding characters, except exponent (key 'e') and modulo (key 'm')
  • Enter keys peform the same function as the '=' operator
